Fix My PC Now - split site build

Files:
- index.html
- laptop-repair.html
- remote-support.html
- mail-in-repair.html
- contact.html
- collection.html
- book.html
- styles.css
- script.js
- assets/

How to run locally:
1) Open the folder in a browser using a local server.
2) Example with Python: python -m http.server 8000
3) Visit http://localhost:8000

SEO notes:
- Titles, descriptions, FAQ schema, canonical links, robots.txt, and sitemap.xml are included.
- The site targets UK computer repair, laptop repair, remote support, mail-in repair, and collection & return search terms.
- Windows PCs and laptops only. No Mac repairs.

Forms:
- Contact and booking forms open your email client using mailto:support@fixmypcnow.co.uk.
- Replace the email address in script.js and page markup if required.
- Stripe payment button is currently a request-to-email action so the link is always working.
